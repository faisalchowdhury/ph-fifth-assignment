Question-3: What is event delegation in the DOM?

Event Delegation is a concept where we apply events in the Parent element instead of applying event on several child , that how we our code become more convenient and faster because now we can set event on all of the child Elements of the Parent using a single Event , and more importantly if we need to add new child element in future automatically new child got the same Event , that how our code will be more dynamic and easy to manage

When any event get fire , in node tree get trough on three thinks , firstly the browser go trough the Capturing phase this time the browser start Capturing every html , body ,element , text ,content all and came to second phase which is Targeting Phase , that time the Event Target the element we have added event , and lastly from the position browser go the again top position and during this time it searches any other trigger event is there on its way or not this phase is Call Bubbling Phase
